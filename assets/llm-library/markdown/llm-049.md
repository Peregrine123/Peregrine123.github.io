## Abstract + Introduction
### 背景
- 基于Softmax的全注意力机制, 是行业共识
- 线性注意力的探索: 为了解决Softmax的N方复杂度, 学界提出了RWKV, Mamba(SSM), RetNet等线性注意力(RNN变体), 他们将复杂度降为N, 推理时只需更新一个固定大小的状态
- 混合模型的尝试: Jamba试图将SSM与Transformer层混合, 试图进行折中, 但是效果不好

#### Softmax Attn
$$\text{Attention}(Q, K, V) = \text{Softmax}\left(\frac{Q K^T}{\sqrt{d_k}}\right) V$$
- 为什么Softmax Attn锁定了$N^2$的复杂度?
	- 在线性代数中, 矩阵乘法满足结合率$(A \times B) \times C = A \times (B \times C)$, 也就是说, 如果$K^T$ 是 $(d, N)$，$V$ 是 $(N, d)$。结果是一个 **$(d, d)$** 的小矩阵, 这一步就是$O(N)$的, 我们用$Q$再去乘这个小矩阵, 那么我们就从始至终规避了$N \times N$ 的大矩阵，复杂度变成了线性的 $O(N)$！
	- 当时Softmax已经规定了我们的计算顺序, 就是必须先算KV, 这导致我们一定要经历$N \times N$ 的大矩阵


### 挑战
- 一直以来, 人们认为: (1)极高的推理吞吐量 (2)低内存占用(KV Cache), (3)媲美全注意力的精细召回能力无法同时兼得
- Agent和RL的需求: 随着模型向Agent和RL发展, 模型需要处理超长的CoT和历史轨迹, Softmax Attention的KV Cache增长过快, 容易OOM, 而线性注意力召回又太差了
### 贡献
>[!核心论点] 
> - 引入细粒度门控的Delta规则和特定的混合架构, Kimi Linear可以在长上下文和强化学习场景, 超越同等规模的全注意力模型(MLA)
> - 提出KDA: 一种线性模块, 结合了DeltaNet的更新规则和GLA的通道级衰减, 解决了记忆容量的问题
> - 3:1混合架构: 大部分层使用KDA, 少量层使用MLA, 抛弃了位置编码, 让KDA承担位置建模的角色

## Method
### Preliminary
- 经典的线性注意力
	- $$S_t = S_{t-1} + k_t v_t^\top$$$$o_t = S_t^\top q_t$$
	- 传统的做法, 就是不断通过叠加的方式更新我们$S_t$,就像我们最原始的RNN
	- 问题就是, 没有引入遗忘机制, 导致所有的信息都直接被叠加在上面, 会造成信息过载
### DeltaNet
- 动机: 不通过盲目累加, 而是根据错误来修正
- 在线梯度下降他把记忆过程当作是一个online learning
	- 目标: 当前的S能够完美的映射$k_t \to v_t$
	- 损失函数$\mathcal{L}_t = \frac{1}{2} \| S^T k_t - v_t \|^2$
- 公式
	- $$S_t = S_{t-1} - \beta_t \nabla \mathcal{L} = (I - \beta_t k_t k_t^\top) S_{t-1} + \beta_t k_t v_t^\top$$
	- 他使得模型具备的更新的能力
### Gated DeltaNet
- 动机: 让模型具备遗忘能力
- 加入一个遗忘门$\alpha_t \in [0, 1]$
- 公式
	- $$S_t = \alpha_t (I - \beta_t k_t k_t^\top) S_{t-1} + \beta_t k_t v_t^\top$$
	- 局限性: 由于$\alpha_t$ 通常是一个**标量**, 所以他是基于整个Head的遗忘, 是粗粒度的
### Kimi DeltaNet
- KDA 在此基础上引入了 $\text{Diag}(\alpha_t)$
	- $$S_t = \text{Optimize}(\text{Decay}(S_{t-1}))$$
	- 相较于之前标量是对句子级别的, $\text{Diag}(\alpha_t)$则是对通道级别的细粒度门控, 让模型得到了更加精准的衰减控制
	- 这种衰减控制, 本事上就是一种位置编码, 因此Kimi大胆的抛弃了RoPE
### NoPE
#### 相对位置的数学形式
- RoPE 通过旋转矩阵 $R$ 来编码相对距离 $(t-i)$：
$$\text{Attention} \propto q_t^T (\prod R) k_i$$
- 而线性注意力的衰减项 $\alpha$，本质上也是在对距离进行指数级编码：
$$\text{State Contribution} \propto q_t^T (\prod_{j=i+1}^t \alpha_j) k_i$$
- 算法优势
	- RoPE是基于固定频率的, 他对距离的感知是硬性的
	- KDA的数据依赖衰减, 则是由当前输入决定的, 因此他不再受限于训练时的固定位置索引
### 3:1混合架构必要性
#### 压缩: KDA 3
- 负责绝大多数的信息处理, 进行有损压缩, 提取语义摘要
#### 检索MLA 1
- 负责关键信息纠错, 进行精确匹配, 利用Softmax Attn进行查漏补缺
