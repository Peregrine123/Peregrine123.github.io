## Abstract + Introduction
### 背景
- LLM推理加速领域, 投机解码(Speculative Decoding, SD)是学术界的热点
- SD的核心思想是通过草稿模型进行快速生成多个tokens, 然后再使用大模型进行并行验证, 接受正确的部分, 从而提高推理效率
- 草稿模型的来源
	- 额外训练一个小模型
	- 直接使用同系列的小模型 比如LLAMA-70B调用LLAMA-7B做草稿模型
### 挑战
- 为了应对传统SD的挑战, 学届推出了self-SD, 利用模型本身的剪枝模型来作为草稿模型
- Self-SD通过稀疏化来构建草稿模型, 比如跳过中间某些层
- 核心挑战: 跳层策略是静态的, 固定的
- 这种静态策略导致草稿模型泛化能力差, 当应对的场景恰好是缺失层具备时, 就会出现准确率下降问题
### 贡献
>[!核心论点] 
> - 利用上一步验证阶段产生的**完整隐藏状态** 作为“真值” (ground truth)，通过**动态规划 (Dynamic Programming)** 算法 ，在**推理时 (on-the-fly)** 为**下一个**草稿步骤实时计算出“最优”的跳层策略。
