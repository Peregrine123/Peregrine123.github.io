## Abstract + Introduction
### 挑战
- 视觉任务中词汇表的缺失: 在NLP中, [[BERT]]的成功是基于预定义的词汇表, 而现有的方法都是直接预测像素值
- 像素级的恢复任务会破坏模型性能
### 贡献
>[!核心论点] 
> - 构建一个图像词汇表, 将图像编码为离散的token $z$, 词汇表大小为8192
> - 预训练任务: 不在是和VQ-VAE相似的重建图像, 而是类BERT的预测掩码ID

## Method
### 预训练阶段
- Masked Image Modeling
	- 图像被分为两个视图, 一个是原始图像的Patch, 另一个是视觉Token
	- 块级掩码: 不但对单个Patch进行掩码, 而是对大的矩阵块进行掩码, 迫使模型理解上下文
	- 流程
		- 图像输入backbone ViT
		- 百分之40的Patch被随机掩盖, 替换为特殊的Embedding符号
		- 模型输出每个掩码位置的Logits, 通过Softmax分类器预测对应的Visual Token
- 微调阶段
	- 预训练后的BEIT编码器可以直接添加任务层进行微调
	- 微调阶段无需生成Visual Token, 只需原始图像作为输入
