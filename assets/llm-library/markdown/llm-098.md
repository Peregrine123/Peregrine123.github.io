## Abstract + Introduction
### 背景
- 技术流派
	- VAE(变分自编码器): 主流的VAE使用连续的潜在变量, 易于训练, 但是生成的图像比较模糊
	- 自回归模型: 通过逐个预测像素或者样本, 生成质量极高, 但是效率很慢, 且缺乏在高层次抽象上进行操作的潜空间
	- GAN: 生成质量高, 但是训练非常不稳定
### 挑战
- 后验坍塌, 将VAE与自回归编码器结合时, 编码器太强了, 以至于把潜在变量忽略掉, 导致无法学到有用的特征
- 数据的本质属性: 许多重要的数据模态本质上是离散的(如语言, 语音, 图像中的物体类别), 但是当时主流是采用连续变量进行建模
- 离散优化的困难: 将离散变量引入通常不可导, 导致无法反向传播
### 贡献
>[!核心论点] 
> - 我们可以通过一种矢量量化(Vector Quantilization, VQ), 在VAE中学到高质量的离散表示, 从而兼具VAE的判断能力以及自回归模型的生成质量
> - 使用了直通估计器, 解决了离散采样不可导的问题, 并且配合codebook机制

## Method
- 动机: 在面对后验坍塌时, 如果潜在空间是离散的, 并强迫编码器必须从有限的codebook中选择code, 那么解码器就无法轻易忽视这些code, 此外, 离散表示能够自然的对数据进行高度压缩和抽象

- 过程
	- 编码, 得到中间变量z
	- 量化: 对于每个中间变量, 他必须在codebook中找到距离最近的向量$e_k$来替换他
		- 这个过程把连续的特征变成离散的索引
	- 解码: 从$e_k$中还原出原始图片

- 反向传播：直通估计器 (Straight-Through Estimator)
	- **问题：** `argmin` 操作是不可导的，梯度断了。
	- **解决：** 作者直接把解码器输入的梯度 $\nabla_{z_q}L$ **原封不动地复制**给编码器的输出 $z_e(x)$。
	- **原理：** 虽然前向传播是离散的，但我们假设量化过程带来的误差很小，所以让编码器朝着“如果我不被量化，应该怎么走”的方向更新是合理的

- 损失函数 (The Loss Function)
$$L = \underbrace{\log p(x|z_q(x))}_{\text{Reconstruction Loss}} + \underbrace{\|sg[z_e(x)] - e\|_2^2}_{\text{Codebook Loss}} + \underbrace{\beta \|z_e(x) - sg[e]\|_2^2}_{\text{Commitment Loss}}$$
- **第一项 (Reconstruction)：** 优化编码器和解码器，让重构图像更清晰。
- **第二项 (Codebook Loss)：** `sg` 代表 Stop Gradient（停止梯度）。这一项**只更新码本向量 $e$**，让码本里的向量主动向编码器输出的特征“靠拢”。
- **第三项 (Commitment Loss)：** 这一项**只更新编码器**。让编码器输出的特征 $z_e(x)$ 不要乱跑，尽量靠近当前的码本向量，防止训练发散。$\beta$ 是一个超参数（论文中设为 0.25)

## Exp
### 音频中的“无监督发现” (Audio - The Highlight)

**这是全篇论文最震撼、最具有“第一性原理”启发的实验。** 它揭示了 VQ-VAE 到底学到了什么。

- **实验设置：**
    - 数据集：**VCTK**（原始语音波形）。
    - 模型架构：类似 WaveNet 的扩张卷积解码器 。
    - 离散空间：潜在空间比原始波形小 64 倍 。
- **结果 1：内容保留，韵律改变 (Reconstruction)**
    - 当重构语音时，**原本的文字内容完全保留**，但**韵律（Prosody，如语调、节奏）发生了改变** 。
    - **导师解读：** 这说明 VQ-VAE 自动过滤掉了波形中的低级细节（语调），只提取了高级的、本质的**语义信息**（说了什么话）。
- **结果 2：说话人转换 (Speaker Conversion)**
    - 解码器被额外输入了一个 Speaker ID（说话人身份）。
    - 实验显示，可以将一个人的语音内容提取出来（Latent），配上另一个人的 ID 进行解码，生成的语音**内容不变，声音变成了另一个人** 。
    - **结论：** 模型成功实现了**内容（离散码）**与**风格（Speaker ID）**的解耦（Disentanglement）。
- **结果 3：无监督发现音素 (Unsupervised Phonemes)**
    - 作者将学习到的离散 Latent 代码与真实的音素标签（Ground Truth）进行比对。
    - **随机 Latent 的准确率：** 7.2%。
    - **VQ-VAE Latent 的准确率：** **49.3%** 。
    - **结论：** 这是一个惊人的数字！模型在完全没有任何语言学监督的情况下，**自己“发明”了一套类似人类音素的符号系统**来描述语音 。
