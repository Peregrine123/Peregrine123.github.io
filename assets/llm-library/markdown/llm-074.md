## Abstract + Introduction
### 背景
- STaR的局限性: 高度依赖精心策划的QA数据集
- 推理具备普遍性: 不仅存在于数学或者逻辑, 他隐含在几乎所有文本的逻辑先接触
### 挑战
- 数据的泛化与规模: 如何让模型在不依赖高质量人工标注推理数据的情况下, 从海量, 非结构的互联网文本学习通用推理
- 计算开销与训练效率: 每个Token在生成推理序列都会带来巨大的计算成本
- 推理的隐匿性: 模型最初并不知道何时该进行内部思考, 也不知道如何利用这些思考来优化预测
