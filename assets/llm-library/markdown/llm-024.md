## Abstract + Introduction
### 背景
- Self-instruct范式: 开发者广泛采用该模式, 通过少量种子指令提示LLM生成更多样化的合成数据, 以解决高质量人咧数据稀缺的问题
### 挑战

### 贡献
>[!核心论点] 
> - 合成数据时, 让模型不经生成问题和答案 更生成规划计划, 可以显著提升合成数据的复杂度和质量
> - 自动数据清洗机制
> - 在推理引导合成上取得很好的效果

## Method
### 概述
- 第一阶段: 给定少量人类标注的种子指令, 我们指导大模型逐步推理, 生成具有相应复杂度的指令
- 合成数据筛选, 保留高质量数据
- 第二阶段: 我们使用合成的数据来训练大语言模型
