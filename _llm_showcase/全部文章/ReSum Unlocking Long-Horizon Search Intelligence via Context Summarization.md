## Abstract + Introduction
### 背景
- 主流范式: ReAct
- 记忆机制: 只增不减, 将所有的历史消息都append到上下文
### 挑战
- 上下文耗尽
- 截断即失忆: 简单的阶段策略(丢弃早期历史)会破坏推理链条, 导致Agent忘记最初的线索或事实
### 贡献
>[!核心论点] 
> - 通过周期性的上下文摘要, 将冗长的交互历史压缩为紧凑的推理状态, 实现理论上的无限步探索
> - ReSum范式: 推理->摘要->重置的范式
> - ReSum-GRPO算法: 引入了优势广播的强化学习算法, 让Agent学会就摘要进行推理
