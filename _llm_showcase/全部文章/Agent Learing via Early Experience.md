## Abstract + Introduction
### 背景
- 主流的技术路线
	- 预训练+SFT, 或者我们说是模仿学习
	- RL
### 挑战
- 模仿学习存在天花板
	- 高质量专家数据昂贵且费时
	- 泛化能力差
- RL并不够稳健
	- 奖励缺失: 有许多领域, 其奖励是无法量化的, 即便可以量化, 但是奖励信号十分稀疏
	- 由于奖励信号的稀疏, 会导致训练困难
- 核心矛盾: 我们既想要RL的可扩展性, 又想要模仿学习的无需设计奖励的特点, 如何得到呢
### 贡献
>[!核心论点] 
> - 作者认为除了模仿学习和RL, 存在第三种范式, 就是早期经验
> - 该范式同样是无奖励的, 但由于自我探索并观察自身行为所产生的反馈, 可以当做是一种监督信号
>

## Method
### 隐式世界建模(Implicit World Modeling, IWW)
- 动机
	- 标准的模仿学习(Intimidate Learning, IL)是盲目的, 他不理解 (a->s) 的状态转移
	- 我们试图强迫模型去理解为什么结果与动作的关系
- 数据构建
	- 从专家数据 $\mathcal{D}_{expert}$ 中取出每一个状态 $s_i$
	- 用 Agent的初始策略 $\pi_{\theta}$（比如 SFT 训练的那个模型）在  上采样，生成 $K$ 个 **“替代动作” (alternative actions)**  。这些动作大概率是“非专家”的、甚至是错误的
	- 在真实环境中**执行**这 $K$ 个替代动作，并记录它们各自导致的 **“结果状态” (resulting next state)** $s_i^j$ 
	- 这就构建了一个新的“体验”数据集 $\mathcal{D}_{rollout} = \{(s_i, a_i^j, s_i^j)\}$
- 训练(阶段一)
	- 训练它一个辅助任务：**给定“当前状态” $s_i$ 和“替代动作” $a_i^j$，请你预测“结果状态” $s_i^j$**
- **训练（阶段二）：**
	- 在第一阶段“预热” (warm-up) 之后，再用标准的 IL 损失（即 $\mathcal{L}_{IL}$）在专家数据 $\mathcal{D}_{expert}$ 上进行微调

### 自我反思 SR
- 动机
	- IWW显式的建模了动作与状态的因果关系, 但还没有教会模型如何交互
	- 让模型通过对比专家数据和自我行为, 得到对比反思
- 数据构建
	- 在状态 $s_i$，我们有：专家动作 $a_i$ 及其后果 $s_{i+1}$
	- 以及Agent 的替代动作 $a_i^j$ 及其后果 $s_i^j$
	- 让教师LLM产生一个prompt
	- 这个 Prompt 会展示 $s_i$、 $a_i$、$s_{i+1}$、$a_i^j$、$s_i^j$(RL五元组)，然后提问：“请分析为什么 $a_i$ 是最好的选择，而 $a_i^j$ 不是？”
	- LLM 会生成一段“反思” $c_i^j$
	- 这就构建了一个新的“反思”数据集 $\mathcal{D}_{refl}$
- 训练
	- 将 $\mathcal{D}_{refl}$ 和 $\mathcal{D}_{expert}$ 混合在一起 。
	- 训练 Agent (LLM) **联合预测**：给定状态 $s_i$，模型必须 **先输出“反思” $c_i^j$，再输出“正确的专家动作” $a_i$**
	- 这是一种[[CoT]]的变体, 但是可以认为是一种判别式的CoT, 显式的将错误与正确的探索路径内置为模型的思考方式, 强调模型需要学习为什么替代方案是错的
